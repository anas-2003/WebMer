## 🛠️ Installation & Setup

### **Method 1: Automated Installation (Recommended)**
```bash
# Clone the repository
git clone https://github.com/anas-2003/WebMer.git
cd WebMer

# Run automated installer
chmod +x install.sh
sudo ./install.sh
```
### **Method 2: Manual Installation**
```bash
# 1. Clone repository
git clone https://github.com/anas-2003/WebMer.git
cd WebMer

# 2. Create virtual environment
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# venv\Scripts\activate  # Windows

# 3. Install system dependencies (Ubuntu/Debian)
sudo apt-get update && sudo apt-get install -y \
    libpcap-dev libffi-dev libssl-dev libxml2-dev \
    libxslt-dev libjpeg-dev zlib1g-dev build-essential \
    wireless-tools aircrack-ng

# 4. Install Python dependencies
pip install -r requirements.txt

# 5. Install as system command
pip install -e .

# 6. Set up permissions for WiFi attacks (Linux)
sudo setcap cap_net_raw+ep $(which python3)
sudo usermod -a -G netdev $USER
```
### **Method 3: Docker Installation**
```bash
# Build Docker image
docker build -t webmer:latest .

# Run WebMer in container
docker run -it --rm --net=host --privileged webmer:latest webmer --help
```
